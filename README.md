# BugZillaScrapper

## Virtual environment

```
python -m venv venv
source venv/bin/activate
```

## Create environment variables

```
cp example.env .env
```

## Install dependencies

```
pip install -r requirements.txt
``` 

## Install CLI

```
pip install -e ./
```

## Using `Issuex CLI`

### Run the tool

```
issuex run
```

### Execute the tool with default settings

```
issuex run:default
```
